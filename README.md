# Swap-Api-Test

## Introduction

Swap-Api-Test project

## Prerequisites

Before you begin, ensure you have [Node.js](https://nodejs.org/) and [Yarn](https://yarnpkg.com/) installed on your machine.

## Installation

To set up the project, follow these steps:

1. Install the dependencies:

    yarn install


2. Start the project:

    yarn dev


4. Run tests

    yarn test
